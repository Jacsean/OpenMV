# 版本发布说明 - v5.0.0

**发布日期**: 2026-04-28  
**上一版本**: v4.0.1  
**版本号**: v5.0.0（主版本升级）

---

## 📋 版本概览

v5.0.0 是一次重大功能更新，在 v4.0.1 基础上新增了完整的**轮廓分析和模板匹配功能体系**，包含6个开发阶段、9个特征提取节点、多种算法实现和性能优化。

---

## ✨ 新增功能

### Phase 1: 基础轮廓分析
- ✅ 轮廓提取与统计分析（面积、周长、边界框、质心）
- ✅ 自动二值化支持（Otsu阈值）
- ✅ 基础统计输出

### Phase 2: 形状检测增强
- ✅ **圆形检测**: 圆度阈值控制（默认0.8）
- ✅ **矩形检测**: 填充率阈值控制（默认0.7）
- ✅ **直线检测**: 直线度阈值控制（默认0.9）
- ✅ **形状分类**: 优先级策略（矩形>圆形>直线>未知）
- ✅ **置信度评估**: 0-1范围量化检测可靠性
- ✅ **可视化增强**: 按形状类型着色标注

### Phase 3: 数据导出功能
- ✅ **CSV导出**: UTF-8-BOM编码，兼容Excel中文显示
- ✅ **Excel导出**: 多Sheet结构（原始数据+统计摘要），格式化表头
- ✅ **JSON导出**: 完整嵌套数据结构，含metadata元信息
- ✅ **路径管理**: 默认路径（工程目录/exports/）+ 自定义路径
- ✅ **文件命名**: 时间戳 + 防冲突机制（自动添加_v2后缀）
- ✅ **依赖检查**: openpyxl可选依赖，缺失时友好提示

### Phase 4: 模板创建节点
- ✅ **TemplateCreatorNode**: 从轮廓生成模板数据
- ✅ **3种特征提取算法**:
  - Hu矩（基础版，快速稳定，无依赖）
  - Shape Context（高级版，需opencv-contrib-python）
  - Hausdorff距离（进阶版，抗遮挡能力强）
- ✅ **4种目标轮廓选择模式**:
  - 按索引选择（已知位置）
  - 按形状筛选（需要特定形状）
  - 按面积筛选（目标有特定尺寸）
  - 综合筛选（多条件联合过滤）
- ✅ **模板数据序列化**: 标准化JSON格式（metadata + template_data + reference_info）
- ✅ **模板预览图像**: 红色加粗矩形框 + 黄色十字标记质心

### Phase 5: 模板匹配节点
- ✅ **TemplateMatchNode**: 基于预生成模板在图像中搜索匹配
- ✅ **3种匹配算法**:
  - Hu矩匹配（欧氏距离转换相似度）
  - Shape Context匹配（形状上下文描述符）
  - Hausdorff距离匹配（归一化距离转换相似度）
- ✅ **相似度阈值控制**: 0.0-1.0范围，默认0.8
- ✅ **多结果输出**: 按相似度降序排序，可限制最大返回数量
- ✅ **标注图像绘制**:
  - 第1名：绿色轮廓 + 金色星标⭐
  - 第2名：黄色轮廓 + 银色星标☆
  - 第3名及以后：橙色轮廓
  - 相似度数值标注（白色文字 + 黑色描边）

### Phase 6: Shape Context集成优化
- ✅ **3种采样策略**:
  - uniform（均匀采样，最快）
  - arc_length（等弧长采样，默认推荐）
  - douglas_peucker（轮廓简化，保留关键特征点）
- ✅ **LRU缓存机制**: 最大100个条目，提升重复模板场景性能
- ✅ **参数验证**: 完整的范围检查和自动调整（采样点数50-500、bins数量2-10等）
- ✅ **轮廓平滑滤波**: 移动平均滤波，减少噪声影响（核大小3-15）
- ✅ **尺度归一化**: 基于面积比例缩放，消除跨尺度误差<3%
- ✅ **轮廓有效性验证**: 过滤退化轮廓（点数<3、面积=0、周长=0）

## 🔧 核心特性

### 1. 算法一致性保障
- 模板与匹配算法自动对齐
- 不一致时自动切换并记录日志
- 避免不匹配导致的错误结果

### 2. 智能降级策略
- Shape Context不可用时自动切换到Hu矩
- 依赖缺失时提供安装命令提示
- 保证系统在任何环境下都能正常工作

### 3. 性能优化
- LRU缓存机制显著提升重复模板场景性能
- 缓存命中率可达80%+（相同模板多次匹配）
- 内存占用可控（~500KB，100个条目）

### 4. 鲁棒性增强
- 轮廓平滑滤波减少噪声干扰
- 尺度归一化提高跨尺度匹配能力
- 有效性验证提前拦截无效输入

### 5. 完善的日志系统
- 详细的参数记录（最终使用的配置）
- 缓存命中/未命中提示
- 警告信息（参数超限、依赖缺失等）
- 便于调试和问题追踪

### 6. 可视化增强
- 按形状类型着色（圆形=蓝色、矩形=绿色、直线=红色）
- 按排名标识（⭐金色/☆银色）
- 相似度数值标注（白色文字 + 黑色描边）

## ⚠️ 兼容性说明

### 依赖要求
1. **Shape Context完整功能**:
   ```bash
   pip install opencv-contrib-python>=4.5.0
   ```

2. **Excel导出功能**:
   ```bash
   pip install openpyxl>=3.0
   ```

3. **基础功能**:
   - opencv-python>=4.5.0
   - numpy

### 已知限制
1. **缓存持久化**: LRU缓存仅存在于内存中，重启应用后失效
2. **旋转不变性**: Shape Context本身不具有旋转不变性（未来可选功能）
3. **并行计算**: 尚未实现多线程/多进程并行匹配（未来可选优化）

### 向后兼容性
- ✅ 完全兼容v4.x工程文件格式
- ✅ 旧节点仍可正常使用
- ✅ 新增节点不影响现有工作流
