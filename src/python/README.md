# 图形化视觉编程系统

基于 **NodeGraphQt** 和 **OpenCV** 的可视化编程框架，类似海康VM、基恩士CV-X、康耐视VisionPro等工业视觉软件的图形化编程体验。

## 🎯 项目简介

这是一个功能强大的图形化视觉编程系统，允许用户通过拖拽节点、连接端口的方式构建图像处理流程，无需编写代码即可实现复杂的视觉检测任务。

### 核心特性

- ✅ **可视化编程**: 拖拽式节点编辑，直观的工作流设计
- ✅ **实时执行**: 一键执行整个节点图，实时查看处理结果
- ✅ **模块化设计**: 每个算法封装为独立节点，易于扩展
- ✅ **属性编辑**: 双击节点即可调整参数，实时预览效果
- ✅ **保存/加载**: 支持将工作流保存为JSON文件，方便复用
- ✅ **完全离线**: 所有处理在本地完成，保护数据隐私

## 🎮 使用说明

### 基本操作流程

#### 1. 添加节点

**方法一：从节点库拖拽**

- 左侧面板显示所有可用节点
- 直接拖拽节点到画布

**方法二：右键菜单**

- 在画布上右键点击
- 选择"创建节点"
- 从列表中选择节点类型

#### 2. 连接节点

- 点击节点的输出端口（右侧）
- 拖动到另一个节点的输入端口（左侧）
- 松开鼠标完成连接

#### 3. 配置参数

- 双击节点打开属性面板
- 在右侧属性面板中调整参数
- 例如：Canny边缘检测的低阈值和高阈值

#### 4. 执行流程

- 点击工具栏的"▶ 运行"按钮
- 或选择菜单"运行" → "执行"
- 查看控制台输出的执行结果

#### 5. 保存/加载

- **保存**: 点击"💾 保存"，选择保存路径（JSON格式）
- **加载**: 点击"📂 加载"，选择之前保存的文件

## 📦 节点说明

### IO节点（输入输出）

#### 图像加载节点

- **功能**: 从文件加载图像
- **输入**: 无
- **输出**: 图像数据
- **属性**:
  - 文件路径: 图片文件的完整路径（**直接输入路径或双击编辑**）
  - **使用提示**: 在文本框中直接粘贴或输入完整的文件路径，例如：`D:\images\test.jpg`

#### 图像保存节点

- **功能**: 将图像保存到文件
- **输入**: 图像数据
- **输出**: 状态信息
- **属性**:
  - 保存路径: 输出文件的路径（**直接输入路径或双击编辑**）
  - 状态: 显示保存结果
  - **使用提示**: 在文本框中输入完整的保存路径，例如：`D:\output\result.png`

---

### 处理节点（算法）

#### 灰度化节点

- **功能**: 将彩色图像转换为灰度图像
- **输入**: 彩色图像
- **输出**: 灰度图像
- **参数**: 无

#### 高斯模糊节点

- **功能**: 对图像进行高斯模糊处理
- **输入**: 原始图像
- **输出**: 模糊后的图像
- **参数**:
  - 核大小 (3-15): 模糊程度，值越大越模糊
  - Sigma X (0-10): 标准差，控制模糊强度

#### Canny边缘检测节点

- **功能**: 使用Canny算法检测图像边缘
- **输入**: 原始图像
- **输出**: 边缘图像
- **参数**:
  - 低阈值 (0-255): 边缘检测的下限
  - 高阈值 (0-255): 边缘检测的上限

#### 二值化节点

- **功能**: 对图像进行阈值二值化处理
- **输入**: 原始图像
- **输出**: 二值化图像
- **参数**:
  - 阈值 (0-255): 二值化的分界点
  - 类型: BINARY, BINARY_INV, TRUNC, TOZERO, TOZERO_INV

---

### 显示节点

#### 图像显示节点

- **功能**: 显示图像信息并提供预览功能
- **输入**: 图像数据
- **输出**: 状态文本（包含图像尺寸、通道数、数据类型等信息）
- **属性**:
  - 状态: 显示图像的详细信息
- **使用方法**:
  1. 将处理后的图像连接到此节点
  2. 运行节点图后，节点会显示图像信息
  3. **双击节点**可打开图像预览窗口
  4. 在预览窗口中可以查看完整图像并保存

**预览窗口功能**:

- ✅ 显示完整尺寸的图像
- ✅ 自动缩放以适应窗口大小
- ✅ 显示图像详细信息（尺寸、通道、数据类型）
- ✅ 支持保存图像到文件
- ✅ 支持滚动查看大图

## 🔧 扩展开发

### 添加新节点类型

#### 步骤1：创建节点类

在 `nodes/processing_nodes.py` 中添加：

```python
from NodeGraphQt import BaseNode
import cv2

class YourCustomNode(BaseNode):
    """
    你的自定义节点
    """
  
    __identifier__ = 'custom'  # 节点分类标识
    NODE_NAME = '你的节点名称'
  
    def __init__(self):
        super(YourCustomNode, self).__init__()
    
        # 添加输入端口
        self.add_input('输入图像', color=(100, 255, 100))
    
        # 添加输出端口
        self.add_output('输出图像', color=(100, 255, 100))
    
        # 添加属性控件
        self.add_slider('param1', '参数1', 0, 100, 50, tab='properties')
    
    def process(self, inputs):
        """
        处理节点逻辑
    
        Args:
            inputs: 输入数据列表
        
        Returns:
            dict: 输出数据字典
        """
        if inputs and len(inputs) > 0:
            image = inputs[0][0] if isinstance(inputs[0], list) else inputs[0]
        
            try:
                # 你的处理逻辑
                param1 = int(self.get_property('param1'))
                result = cv2.your_function(image, param1)
            
                return {'输出图像': result}
            except Exception as e:
                print(f"处理错误: {e}")
                return {'输出图像': None}
    
        return {'输出图像': None}
```

#### 步骤2：注册节点

在 `ui/main_window.py` 的 `_register_nodes()` 方法中添加：

```python
from nodes.processing_nodes import YourCustomNode

def _register_nodes(self):
    # ... 现有节点注册
  
    # 注册你的新节点
    self.node_graph.register_node(YourCustomNode)
```

#### 步骤3：重启程序

重新运行 `python main.py`，新节点将出现在节点库中。

### Q: 节点连接后无法执行?

1. 确保所有节点都已正确连接
2. 确保"图像加载"节点设置了正确的文件路径

- 添加"图像显示"节点连接到流程末尾
- 或在"图像保存"节点设置保存路径
- 执行后查看控制台输出

### Q: 如何调试节点?

1. 在节点的 `process()` 方法中添加 `print()` 语句
2. 查看控制台输出
3. 检查输入数据是否正确

### Q: 支持哪些图像格式?

OpenCV支持的格式均可：

- JPG/JPEG
- PNG
- BMP
- TIFF
- WEBP
