# OpenCV视觉系统 - Python版本

## 功能特性

- ✅ 支持常见图片格式 (JPG, PNG, BMP, TIFF, WEBP)
- ✅ 多种图像处理算法
  - 灰度化
  - 高斯模糊
  - 中值滤波
  - Canny边缘检测
  - 二值化/自适应二值化
  - Sobel/Laplacian边缘检测
  - 形态学操作(膨胀/腐蚀)
  - 直方图均衡化
- ✅ 鼠标交互ROI选择与裁剪
- ✅ 可调节算法参数
- ✅ 完全离线运行

## 环境要求

- Python 3.7+
- OpenCV 4.5.0+
- Pillow 8.0+

## 安装依赖

```bash
pip install -r requirements.txt
```

## 运行程序

```bash
python vision_system.py
```

## 使用说明

1. **打开图片**: 点击"打开图片"按钮,选择图片文件
2. **应用算法**: 在左侧控制面板选择想要的图像处理算法
3. **调整参数**: 拖动滑块调整算法参数(如Canny阈值、二值化阈值)
4. **ROI操作**:
   - 点击"选择ROI"按钮
   - 在图像上拖动鼠标选择区域
   - 点击"裁剪ROI"进行裁剪
   - 或点击"取消选择"取消操作
5. **保存图片**: 点击"保存图片"按钮保存处理后的图片

## 注意事项

- 所有处理均在本地完成,无需网络连接
- 支持实时预览处理效果
- ROI选择时显示红色矩形框
