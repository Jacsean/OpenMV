# 图形化视觉编程系统 - 快速入门指南

## 🎉 恭喜！系统已成功运行

你现在看到的是类似海康VM、基恩士CV-X的图形化视觉编程界面。

---

## 📖 基本操作

### 1. 添加节点

**方法一：从节点库拖拽**
- 左侧"节点库"面板显示所有可用节点
- 直接拖拽节点到中间画布

**方法二：双击节点库**
- 在节点库中双击节点名称
- 节点会自动添加到画布中央

### 2. 连接节点

1. 将鼠标移到节点的**输出端口**（右侧圆点）
2. 按住鼠标左键拖动
3. 连接到另一个节点的**输入端口**（左侧圆点）
4. 松开鼠标完成连接

### 3. 配置节点参数

1. **选中节点**：点击节点
2. **查看属性**：右侧"属性面板"会显示节点的参数
3. **调整参数**：
   - 文本框：直接输入值
   - 滑块：拖动调整数值
   - 下拉菜单：选择选项
4. **实时生效**：修改参数后，下次执行时会使用新值

### 4. 执行流程

1. 构建好节点流程后
2. 点击工具栏的 **"▶ 运行"** 按钮
3. 查看控制台输出的执行结果

### 5. 保存/加载工作流

**保存：**
- 点击 **"💾 保存"** 按钮
- 选择保存位置
- 输入文件名（.json格式）
- 点击保存

**加载：**
- 点击 **"📂 加载"** 按钮
- 选择之前保存的.json文件
- 工作流会自动恢复

## 🧪 第一个示例：边缘检测

让我们创建一个简单的边缘检测流程：

### 步骤1：添加节点

1. 从节点库拖拽以下节点到画布：
   - **图像加载**（IO分类）
   - **灰度化**（处理分类）
   - **Canny边缘检测**（处理分类）
   - **图像保存**（IO分类）

### 步骤2：配置节点

1. **图像加载节点**：
   - 选中节点
   - 在右侧属性面板的"文件路径"中输入图片路径
   - 例如：`D:\test.jpg`

2. **Canny边缘检测节点**：
   - 选中节点
   - 调整"低阈值"（建议50）
   - 调整"高阈值"（建议150）

3. **图像保存节点**：
   - 选中节点
   - 在"保存路径"中输入输出路径
   - 例如：`D:\output.jpg`

### 步骤3：连接节点

按以下顺序连接：
```
[图像加载] → [灰度化] → [Canny边缘检测] → [图像保存]
```

操作方法：
- 从"图像加载"的输出端口拖到"灰度化"的输入端口
- 从"灰度化"的输出端口拖到"Canny边缘检测"的输入端口
- 从"Canny边缘检测"的输出端口拖到"图像保存"的输入端口

### 步骤4：执行

1. 点击工具栏的 **"▶ 运行"** 按钮
2. 查看控制台输出
3. 检查保存的图片文件

## 💡 提示与技巧

### 画布操作
- **缩放**：鼠标滚轮
- **平移**：按住空格+鼠标拖动，或中键拖动
- **选择多个节点**：框选或Ctrl+点击
- **删除节点**：选中后按Delete键

### 节点操作
- **移动节点**：拖动节点
- **复制节点**：Ctrl+C / Ctrl+V
- **撤销操作**：Ctrl+Z
- **重做操作**：Ctrl+Y

### 连接线
- **删除连线**：点击连线后按Delete
- **重新连线**：删除旧连线后重新连接

## 🐛 常见问题

### Q: 如何知道节点是否连接成功？
A: 连接成功后，两个端口之间会出现一条彩色的连线。

### Q: 执行后没有输出？
A: 检查：
1. 图像加载节点的文件路径是否正确
2. 所有节点是否都已正确连接
3. 查看控制台的错误信息

### Q: 如何查看可用的节点？
A: 左侧的"节点库"面板列出了所有可用节点，按类别分组。

### Q: 可以添加自定义节点吗？
A: 可以！参考README.md中的"扩展开发"章节。
